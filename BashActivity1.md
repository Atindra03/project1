Basic Linux commands:

1)ls -a
Description: display information about their files
ls -A
Description: display information about almost all files


2)ls -b
Description: Lists the names of files in a particular Unix directory
ls -B
Description:Lists the names of files in a particular Unix directory


3)ls -c
Description: lists the contents of your current working directory
ls -C
Description: list entries by columns
 

4)ls -d
Description: list information about files and directories within the file system
ls -D
Description: generate output designed for emacs dired mode


5)ls -f
Description: to list files
ls -F
Description: Appends a character to indicate file type


6)ls -g
Description: group directories before files
ls -G
Description: group directories before files in a long listing, doesn't print group name


7)ls -h
Description: lists files and directories 
ls -H
Description: follow symbolic links listed on the command line

8)ls -i
Description: Displays the inode number of each file or directory.
ls -I
Description: do not list implied entries matching shell pattern
 

9)ls -k
Description: default to 1024byte blocks for file system usage


10)ls -l
Description: list information about files and directories within the file system
ls -L
Description:when showing file information for a symbolic link, show information for the file the link references rather than for the link itself


11)ls -m
Description: list information about files and directories within the file system


12)ls -n
Description: list information about files and directories within the file system
ls -N
Description: print entry names without quoting 


13)ls -o
Description: do not list group information


14)ls -p
Description: append or indicator to directories


15)ls -q
Description: prints eaverything except non-graphic characters
ls -Q
Description: enclose entry names with double quotes


16)ls -r
Description: Reverses the order of sorting
 

17)ls -R
Description: Lists the names of files in a particular Unix directory


18)ls -s
Description: list information about files and directories within the file system
ls -S
Description: prints the allocated size of each file in blocks


19)ls -t
Description: Sorts files by modification time, with the newest files listed first


20)ls -u
Description: sort by and show access time
ls -U
Description: doesn't sort , list entries in directory order

21)ls -v
Description: natural sort of version within text


22)ls -w
Description: list the contents of a directory


23)ls -x
Description: list entries instead of by columns
ls -X
Description: sort alphabetically by entry extension


24)ls -z
Description: print any security content of each file
